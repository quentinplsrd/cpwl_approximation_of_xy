# cpwl_approximation_of_xy


## Setup

### With `pip`
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install -e .
```

### With `uv`

```
uv venv
source .venv/bin/activate
uv add -r requirements.txt
uv pip install -e .
```

## Running TempRegPy model test
```
python scripts/script-run_tempregpy_sims.py
```
